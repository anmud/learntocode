## Aling-self property

We can control flex items individually.

* auto
* flex-start
* flex-end
* center
* baseline
* stretch

Override the align-items on a per flex item basis

## Order property

```css
div:nth-of-type(3n) {
   order: -1;
}
```

The default value is 0. Anything lower will come before those without set values. Anything above will come after.
